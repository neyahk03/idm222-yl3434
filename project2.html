Generic Website

Project overview

As part of our User Interface I class assignments, we were tasked with redesigning a generic website in two specific styles - Grunge/Graffiti and Brutalism. The objective was to improve our research and design skills so that the final website not only conveyed the desired vibe but also ensured a good user experience.

My role: Interaction design, Prototyping, Usability Testing, Visual Design.

Tools: Figma
